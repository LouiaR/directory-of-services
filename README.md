This is the Directory of Services project live at [https://dos.codeyourfuture.io](https://dos.codeyourfuture.io)

## Getting started

Install project dependencies:

```sh
yarn
```

Run project locally:

```sh
cd frontend && yarn start
cd backend && yarn start
```
